# Minkowski

![alt text](screenshot)

Visualization of a worldline through 2+1 Minkowski space.

## Controls

- <kbd>W</kbd>/<kbd>S</kbd> Boost right/left
- <kbd>A</kbd>/<kbd>D</kbd> Rotate
- <kbd>Q</kbd>/<kbd>E</kbd> Boost forward/back
